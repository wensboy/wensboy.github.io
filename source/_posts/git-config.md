## 下载并安装git

前往[Git官网](https://git-scm.com)点击下载进入下载页面，根据自己的操作系统下载安装即可。

**使用如下command查看是否成功安装git:**

```shell
git --version
```

{% note info %}
提示：以下假设在设备上成功下载安装git,并配置好了相关环境变量。
{% endnote %}
